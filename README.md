# ndrzet.github.io

This is the personal portfolio website of Karim Sokolov, a creative developer and web designer from Saint Petersburg, Russia.

## About
This repository contains the source code for my personal portfolio website, showcasing my skills, projects, and interests as a web developer and designer.

## Features
- Modern, responsive design with Material Design aesthetics
- Dark/light theme toggle with persistent user preference
- Smooth animations and transitions
- Mobile-friendly navigation with modal menu
- Intersection Observer for scroll animations

## Technologies Used
### Frontend
- HTML5
- CSS3 (with custom properties and animations)
- JavaScript (ES6+)
- Google Fonts (Roboto)

### Development Tools
- Git for version control
- GitHub Pages for hosting

## Project Structure
```
ndrzet.github.io/
├── index.html          # Main HTML file
├── styles.css          # Stylesheet with all CSS rules
├── script.js           # JavaScript file with all functionality
├── tools.html          # Additional tools page
├── favicon.ico         # Website favicon
├── CNAME               # Custom domain configuration
└── README.md           # This file
```

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/ndrzet/ndrzet.github.io.git
   ```
2. Open `index.html` in a web browser to view the website locally.

## Deployment
The website is automatically deployed via GitHub Pages at https://ndrzet.github.io
